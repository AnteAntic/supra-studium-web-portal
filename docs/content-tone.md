## Core Brand Philosophy

Supra Studium does not sell techniques. It sells the capacity for clinical thinking that becomes visible through hands.

Brand communicates by informing, showing, demonstrating. It does not motivate, transform, or inspire in the coaching sense. Trust comes from precision, not warmth. If text sounds like it wants the reader to feel good about themselves — it is not Supra copy.

**Central tension:** "approach vs. guessing" — the only tension that may drive the narrative.

---

## Three Brand Voices

### Voice 1 — Therapist-Educator

When to use: Educational content, course pages, clinical posts
Tone: Observational, clinically precise, colleague-from-practice. Focus on the case/mechanism, not on the person.

### Voice 2 — Organiser-Activator

When to use: Event announcements, enrollment, calls to apply
Tone: Direct, concrete, calm enthusiasm without hype.

### Voice 3 — Institution

When to use: Website copy, program descriptions, FAQ, logistics
Tone: Professional, clear, no personal references. Information without decoration.

**Rule:** One piece of content = one voice. Never mix.

## Authority Model

| "Technique" positioning | "Clinical thinking" positioning |
|-------------------------|---------------------------------|
| "You'll learn X technique" | "You'll learn when X — and when not X" |
| "Treatment that works" | "An approach that can be repeated" |
| "Results from session 1" | "Understanding that lasts" |
| "Add a tool to your toolkit" | "Change how you read the patient" |

Authority is demonstrated, not declared. Never write "we develop clinical thinking" — show it through content that displays clinical thinking in action.

## Claim Certainty System

### Forbidden register (absolute claims)

- "Eliminates pain" / "Guarantees results" / "Resets the nervous system"
- "Transforms posture" / "From session 1" / "Always works"
- "The only approach that..."

### Allowed register (observational/practical)

- "We often see improvement" / "In practice, with many patients"
- "Can help in cases where..." / "Tension decreases"
- "This region frequently participates in the pain pattern"
- "Worth checking in differential assessment"

### Observational vocabulary (use freely)

`often` · `in practice` · `with many patients` · `can help` · `we often see` · `worth checking` · `in cases where` · `one possible cause` · `part of differential thinking` · `gradually`

Croatian: `često` · `u praksi` · `kod mnogih pacijenata` · `može pomoći` · `često vidimo` · `vrijedi provjeriti`

## Forbidden Phrases

### Marketing language (never use)

- transformiraj / transformira
- moć dodira
- instant rezultat
- osiguraj mjesto odmah
- ostala X mjesta
- investiraj u sebe
- premium
- ekskluzivno
- breakthrough
- life-changing
- revolucionarni pristup
- jedinstven u Hrvatskoj

### Forbidden openings

- "Ovo me promijenilo."
- "Nisam mogao vjerovati."
- "Zamislite..."
- "U današnjem dinamičnom svijetu..."

### Forbidden CTA phrases

- "Ne propusti!" / "Javi se ODMAH!" / "Ovo je zadnja šansa!"
- "Investiraj u sebe!" / "Postani bolji terapeut!"
- "Mjesta lete!"

### Forbidden rhythm pattern

Fragment. Fragment. Fragment. (three-fragment triplet) — always merge into natural sentences.

## Allowed Clinical Vocabulary

Technical terms that signal professional register. Use them as-is, without explanation unless the content is specifically introductory:

`palpacija` · `reakcija tkiva` · `napeta traka` · `trigger točka` · `obrazac boli` · `ishemijski pritisak` · `PIR (post-izometrička relaksacija)` · `lokalna točka` · `distalna točka` · `procjena prije tretmana` · `subokcipitalna regija` · `diferencijalna procjena` · `tkivni odgovor`

## Headline Principles

- Mixed case, never ALL CAPS
- Confident and smaller rather than large and overreaching
- Clinical positioning, not a marketing promise
- Ends the thought — does not tease, does not promise
- Periods at end of observation headings: "Najčešća pitanja prije početka." (FAQ heading rule)
- No em-dashes in headlines (em-dashes banned site-wide; use commas, colons, or parentheses)

### Examples of Supra headlines

```
✅ "Kada znaš da si na pravom mjestu."
✅ "Akupresura nije relaksacija."
✅ "Tkivo govori. Palpacija je slušanje."
✅ "Pet stupnjeva jedne metode."
❌ "Transformiraj svoju terapijsku praksu!"
❌ "JEDINSTVEN PROGRAM U HRVATSKOJ"
❌ "Otkrij moć dodira — postani ekspert!"
```

## CTA Philosophy

### CTA Mode A — Educational moment (no transaction)

```
OK:  "Info o sljedećem ciklusu: tally.so/r/wA5kvD"
OK:  "Više o [temi] — link u bio."
NOT: "Prijavi se dok ima mjesta!" / "Ne propusti ovu priliku!"
```

### CTA Mode B — Pre-enrollment

```
OK:  "Sljedeći termin: [datum], Zagreb. Prijave: tally.so/r/wA5kvD"
OK:  "Mjesta su ograničena — radimo u malim grupama. Prijave otvorene."
NOT: "Mjesta lete!" / "Investiraj u sebe — odmah!"
```

### Standard CTA block (for course pages, social posts)

```
Prijavi se: tally.so/r/wA5kvD
WhatsApp: wa.me/385958558953
Zagreb | @suprastudium
```

**Rule:** One CTA per post or section. A post without a CTA is a legitimate format.

## Page Structure (ideal, non-funnel)

1. **Hero** — clinical positioning, not a promise
2. **Brief observation** — real problem in practice
3. **"What you're starting to notice"** — observation cards, not benefit cards
4. **Clinical snapshots** — 2–3 concrete cases
5. **Method** — technique with mechanism, not just features
6. **Day-by-day program** — concrete, no poetic section names
7. **Instructor** — philosophy, not ego biography
8. **Price and date** — clean, no "invest in yourself"
9. **FAQ** — eyebrow "Česta pitanja", heading ends with period
10. **CTA** — calm, professional

## Subheadline Principles

The hero subheadline acknowledges where the therapist **already is** — not where they will go. This is fundamental. It does not promise transformation. It recognises a state that already exists.

```
✅ "Rad koji ne nagađa."
✅ "Tečaj za terapeute koji već rade i žele razumjeti zašto nešto ne funkcionira."
❌ "Postani stručnjak za trigger point metodu!"
❌ "Naučit ćeš sve o akupresuri."
```

## AI Decision Tree (for generating any output)

```
1. Content type? → MODE A (education) / B (enrollment) / C (event) / D (alumni) / E (info)
2. Voice? → A/D = Voice 1 | B = Voice 1→2 | C = Voice 2 | E = Voice 3
3. Audience layer? → default Layer 2 (experienced practitioner, 3+ years)
4. Platform? → FB (longer) / IG (compressed) / Web (Voice 3 dominant)
5. CTA type? → A = informational or none | B = direct enrollment
6. Claim certainty check → absolute claim? → replace with observational register
7. Negative space → too many conclusions/CTAs? → remove the last sentence
```
